\# Face Recognition Attendance System



A full-stack AI-powered Face Recognition Attendance Web Application built using:



\- Python (Flask)

\- OpenCV (LBPH Algorithm)

\- JavaScript MediaDevices API

\- Bootstrap UI



\## 🚀 Features



\- Automatic Face Registration (25 images auto-capture)

\- Real-time Face Recognition

\- Attendance Marked Popup Notification

\- IN/OUT Time Tracking

\- Excel Export (Daily \& Date Range)

\- Admin Login System

\- Clean Glass UI Design



\## 🛠 Technologies Used



\- Flask

\- OpenCV

\- NumPy

\- Pandas

\- JavaScript

\- Bootstrap
